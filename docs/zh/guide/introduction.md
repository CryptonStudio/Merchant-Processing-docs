## 核心概念

### 商户 (Merchant)
商户是使用我们支付服务的业务实体。每个商户都有唯一的标识符和 API 密钥。

### 地址 (Address)
支付地址是用于接收加密货币的区块链地址。我们为每笔交易生成唯一的地址。

### 网络 (Network)
区块链网络，如以太坊、比特币等。每个网络都有其特定的特性和支持的币种。

### 币种 (Coin)
支持的加密货币，包括原生币种（如 BTC、ETH）和代币（如 USDT、USDC）。

### 交易 (Transaction)
在区块链上的资金转移记录，包括发送方、接收方、金额等信息。

### 提现 (Withdrawal)
将资金从平台转出到外部地址的操作。
